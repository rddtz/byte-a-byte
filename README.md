# Byte a Byte — Site oficial

Este repositório contém o código-fonte do site do **Byte a Byte**, projeto de extensão do Instituto de Informática da UFRGS.

## Sobre o projeto

O Byte a Byte é um curso de introdução à programação voltado para pessoas que nunca tiveram contato com código. O objetivo não é formar programadores profissionais, mas mostrar que qualquer pessoa pode dar os primeiros passos no mundo da tecnologia.

O curso usa **Python** — uma das linguagens mais acessíveis do mercado — e é completamente **gratuito e online**.

### Estrutura do curso

- **Aula 01 — Tipos de dados:** inteiros, decimais, textos e booleanos (`int`, `float`, `str`, `bool`)
- **Aula 02 — Condicionais e laços:** tomada de decisão e repetição (`if/else`, `for`, `while`)
- **Aula 03 — Listas e funções:** organização de dados e reutilização de código (`list`, `def`, `return`)
- **Exercícios semanais** com monitoria disponível todos os dias da semana
- **Avaliação final** para consolidar o aprendizado

### Quem faz

Somos alunos do **Instituto de Informática da UFRGS** e alumni do **Amanhã Tech**, programa do Fundo Amanhã. Acreditamos que a tecnologia deve ser acessível a todos — e o Byte a Byte nasce desse desejo.
